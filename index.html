Game Rules

The goal is to reach the tile with the number 2048 by following these rules:

  • Combine tiles with the same number to create a new one with double the value
  • Each move spawns a new tile (2 or 4)
  • You win when you create a tile with 2048
  • You lose if no more moves are possible
  • You can keep playing after winning to reach higher numbers

How to Play

Use the arrow keys to move all the tiles:

  • All tiles move in the selected direction
  • Matching tiles merge when they collide
  • Plan your moves to create combo chains
  • Keep your highest numbers in a corner
